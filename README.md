### Colemak and Vim and friends

 * **h, j, k, l** - colemak h, n, e and i: movement.
 * **k**: next search result.
 * **K**: previous search result.
 * **u**: go to insert mode.
 * **U**: go to insert mode at begging of current line.
 * **l**: Insert mode.
